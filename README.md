**Diecast Hub India** is a premium, centralized directory for diecast model car collectors in India. Built with a focus on high-performance aesthetics and real-time utility, it allows collectors to discover, filter, and track their favorite stores.

---

## ✨ Features

- **🎯 Precision Filtering**: Search by store name, category, or specific collection tags (e.g., Mini-GT, Hot Wheels, Inno64).
- **📌 Pinned Favorites**: Save your go-to stores. Pins are persisted locally so they are there when you come back.
- **🏎️ Premium UI**: "Midnight Racing" theme featuring glassmorphism, smooth animations, and high-contrast visuals.
- **🔐 Admin Dashboard**: Secure Firebase Auth-powered panel to manage listings in real-time.
- **⚡ Shimmer States**: Smooth loading experience with custom skeleton components.
- **📱 Ultra Responsive**: Optimized for every screen, from small mobiles to large desktops.

---

## 🛠️ Tech Stack

- **Frontend**: React 19 (TypeScript)
- **Styling**: Tailwind CSS (Custom Design System)
- **Database**: Firebase Cloud Firestore

- **Icons**: React Icons (Fa, Fi, Bs, Si)

## 💻 Local Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/Javeed726/diecast-hub-india.git
   cd diecast-hub-india
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root and add your Firebase credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_key
   VITE_FIREBASE_AUTH_DOMAIN=your_domain
   VITE_FIREBASE_PROJECT_ID=your_id
   VITE_FIREBASE_STORAGE_BUCKET=your_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```

---

## 📦 Deployment (Vercel)

1. Push your code to GitHub.
2. Import project in [Vercel](https://vercel.com).
3. **IMPORTANT**: Add your `.env` variables in **Settings > Environment Variables** on the Vercel dashboard.
4. Deploy!
